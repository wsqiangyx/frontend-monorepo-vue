## 阅读顺序建议

如果你是模板消费者：

1. 先看根 `README.md`
2. 再看根 `TEMPLATE.md`
3. 再看 `docs/总体设计/Vue3 中后台前端平台 Monorepo 架构设计方案.md`
4. 最后按本目录中的 [模板初始化与裁剪指南.md](./模板初始化与裁剪指南.md) 执行

如果你是维护者，要从 0 理解分阶段详细设计：

1. 先看根 `AGENTS.md`
2. 再看 `docs/总体设计/Vue3 中后台前端平台 Monorepo 架构设计方案.md`
3. 最后看 `docs/专题文档/README.md`

## 使用约定

- 教程文档回答“怎么做”，不重复维护仓库约束正文。
- 若涉及模板正式规则、初始化步骤、环境变量样例、阶段目标或公共契约变更，优先同步根 `README.md`、根 `AGENTS.md`、根 `TEMPLATE.md` 与总体设计主文档。

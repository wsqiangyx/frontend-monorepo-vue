## 一、项目上下文

### 源仓库（yudao-ui-admin-vue3）

- **完整路径**：`D:\workspace\front-monorepo-vue\docs\治理\yudao-ui-admin-vue3-master`
- **2067 个源文件**，Element Plus 2.4.2 + Vue3 3.3.8 + Vite4 + Pinia
- 完整的芋道后端配套前端，功能矩阵覆盖 20+ 业务模块
- 响应格式：`{ code, msg, data }`
- 功能清单详见 `D:\workspace\front-monorepo-vue\docs\治理\yudao-ui-admin-vue3-master/README.md`

### 目标仓库（本 Monorepo）

- **TDesign Vue Next** (非 Element Plus)，pnpm workspace Monorepo
- 依赖方向：`yudao-app → shared-ui → shared-service → shared-utils`
- MSW Mock 驱动开发，后端无关
- 共享层：`design-tokens`, `shared-utils`, `shared-i18n`, `shared-service`, `shared-ui`, `shared-workflow`

### 迁移原则

1. **技术栈适配**：Element Plus → TDesign（同名组件映射，缺失组件自研或用 shared-ui 封装）
2. **架构对齐**：API 进 shared-service/modules/，组件进 shared-ui/，页面进 apps/vue3-app/src/views/
3. **渐进迁移**：先核心基座（系统管理），再通用模块（基础设施+工作流），最后业务模块
4. **Mock 先行**：每个模块先建 MSW handlers，保证不依赖后端可独立开发验证

## 二、迁移批次规划

### Phase 1：平台基座增强（迁移源仓库核心架构）

**目标**：补齐登录、布局、路由、权限体系

### Phase 2：系统管理模块（必选核心）

**目标**：用户/角色/菜单/部门/岗位/字典/租户/短信/通知等

### Phase 3：基础设施模块（必选通用）

**目标**：代码生成/文件管理/定时任务/配置管理/API日志等

### Phase 4：工作流模块（可选通用）

**目标**：BPMN 设计器/Simple 设计器/流程引擎/任务管理

### Phase 5+：业务模块（按需迁移）

**目标**：支付/报表/大屏/商城/CRM/ERP/MES/WMS/AI/IoT 等

### 5.2 代码放置规则

| 内容类型                 | 放置位置                                                            |
| ------------------------ | ------------------------------------------------------------------- |
| API 封装（接口调用函数） | `packages/shared-service/src/modules/{module}/`                     |
| MSW Mock handlers        | `packages/shared-service/src/mock/handlers/{module}/`               |
| TypeScript 类型定义      | `packages/shared-service/src/types/` 或对应模块目录                 |
| 通用 Vue 组件            | `packages/shared-ui/src/components/`                                |
| 纯函数工具               | `packages/shared-utils/src/`                                        |
| Vue Hooks/Composables    | `packages/shared-ui/src/composables/`（依赖 Vue）                   |
| 页面组件                 | `apps/vue3-app/src/views/{module}/`                                 |
| Pinia Store              | `apps/vue3-app/src/stores/`                                         |
| 路由配置                 | `apps/vue3-app/src/router/modules/`                                 |
| 指令                     | `apps/vue3-app/src/directives/`                                     |
| 国际化文案               | `apps/vue3-app/src/locales/` 或 `packages/shared-i18n/src/locales/` |
| 静态资源（SVG/图片）     | `apps/vue3-app/src/assets/`                                         |
